Homey WooCommerce addon for Homey theme only.

== Description ==
This plugin provides WooCommerce integration with [Homey](https://gethomey.io/) theme only.

Using this plugin **Homey** agents can buy membership and pay for listings using **WooCommerce** payment gateways.

This plugin is also translation ready and you can translate it to your language using pot file provided in languages directory.

== Frequently Asked Questions ==

= Will this addon work with other realestate themes? =

No, this is only for Homey theme.


== Installation ==
1. Unzip the downloaded package
1. Upload `houzez-woo-addon` to the `/wp-content/plugins/` directory
1. Activate the plugin through the \'Plugins\' menu in WordPress
